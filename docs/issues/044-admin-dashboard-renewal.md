# 044 — 어드민 대시보드 리뉴얼

## 배경
기존 대시보드가 3개 섹션(출근 현황, 보건증 주의, 바로가기 3개)으로 단순해서 크레딧/티어 시스템, 추가근무/대타 처리, 주간 스케줄 등 핵심 운영 정보를 한눈에 파악할 수 없었음.

## 변경 내용

### 리뉴얼된 대시보드 구성 (8개 섹션, 우선순위순)

1. **헤더** — 매장별 현재 근무 인원 인라인 표시
2. **KPI 요약 카드** (NEW) — 출근율/지각/미출근/처리필요 4장, 색상 코딩
3. **처리 필요 배너** (NEW) — 추가근무+대타+보건증 통합 알림, 조건부 렌더링
4. **오늘 출근 현황** (개선) — 매장별 그룹핑 + 상태 요약 도트 추가
5. **팀 크레딧 현황** (NEW) — 평균 점수, 티어 분포, 최고 스트릭, 위험 직원
6. **이번 주 스케줄** (NEW) — 7일 캘린더 스트립, 0명 경고
7. **빠른 이동 그리드** (개선) — 3개→6개 타일, 배지 표시
8. **최근 활동** (NEW) — 최근 5개 알림 인라인
