# 037 — 어드민 체크리스트 리뉴얼

## 배경

기존 `/admin/checklists` 페이지가 관리하기 불편하다는 피드백.
- 항목 수정 불가 (삭제 후 재생성만 가능)
- 어떤 근무지/포지션에 항목이 할당됐는지 한눈에 파악 불가
- 직원이 실제로 보게 될 화면을 어드민이 미리 확인할 수 없음
- 항목 순서 변경 불가

## 수정 내용

기존 출근/퇴근 탭 + 전체 목록 방식 제거.
**조합별 모아보기**와 **미리보기** 두 뷰로 전면 교체.

### 뷰 모드

| 뷰 | 설명 |
|----|------|
| 조합별 모아보기 | 근무지×포지션 조합별 아코디언 카드. 안에서 추가·수정·삭제·순서변경·토글 모두 처리 |
| 미리보기 | 근무지·포지션 선택 → 직원이 보게 될 체크리스트 시뮬레이션 |

### 조합별 모아보기
7개 조합 카드 (전체 적용 / 카페 공통 / 카페·홀 / 카페·주방 / 카페·쇼룸 / 공장 / 케이터링)
- 각 카드에 출근/퇴근 항목 수 배지
- 펼치면 출근·퇴근 섹션 분리, 항목 인라인 수정·삭제·토글·순서 변경
- 각 섹션 하단에 "항목 추가하기" 버튼

### 미리보기
- 근무지 + 포지션 드롭다운
- 출근/퇴근 탭
- 실제 ChecklistSheet UI로 시뮬레이션 (저장 없음, 초기화 가능)
