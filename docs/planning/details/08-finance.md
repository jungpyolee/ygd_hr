### 3-3. ARPU (Average Revenue Per User) 계산

> "User" = 유료 매장 1개 (조직 단위 과금)

**플랜 믹스 가정 (정식 출시 6개월 후):**

| 플랜 | 비율 | 월 매출 기여 |
|------|------|-------------|
| Free | 75% | 0원 |
| Starter | 20% | 29,900원 |
| Pro | 5% | 79,900원 |

```
ARPU (전체) = (0.75 x 0) + (0.20 x 29,900) + (0.05 x 79,900)
            = 0 + 5,980 + 3,995
            = 9,975원/월 (전체 가입 매장 대비)

ARPU (유료만) = (0.80 x 29,900) + (0.20 x 79,900)
              = 23,920 + 15,980
              = 39,900원/월 (유료 매장 대비)
```

### 3-4. LTV (Lifetime Value) 추정

```
LTV = ARPU(유료) x 평균 구독 기간(월)

평균 구독 기간 = 1 / 월간 이탈률
  보수적: 1 / 0.08 = 12.5개월
  중립:   1 / 0.05 = 20개월
  낙관:   1 / 0.03 = 33.3개월

LTV (보수적) = 39,900 x 12.5 = 498,750원
LTV (중립)   = 39,900 x 20   = 798,000원
LTV (낙관)   = 39,900 x 33.3 = 1,328,670원
```

### 3-5. CAC (Customer Acquisition Cost) 목표

```
건전한 SaaS 비율: LTV / CAC >= 3

CAC 목표 (중립 LTV 기준):
  798,000 / 3 = 266,000원 이하

현실적 CAC 채널별 예측:
  - 네이버 블로그/카페 홍보 (무료)     : CAC 0원 (시간 투입만)
  - 인스타그램 F&B 커뮤니티             : CAC 5,000~15,000원
  - 네이버 키워드 광고                  : CAC 30,000~80,000원
  - 입소문/소개 (사장님 네트워크)       : CAC 0원
  - 카카오 비즈보드 광고                : CAC 20,000~50,000원
```

**초기 전략: CAC 최소화**
- 연경당을 레퍼런스 케이스로 활용
- F&B 사장님 커뮤니티(네이버 카페, 인스타그램)에서 무료 베타 모집

### 4-2. BEP 매장 수 계산

**공헌이익 (Contribution Margin):**


**BEP = 고정비 / 가중평균 공헌이익**

1. 예비창업패키지 또는 초기창업패키지 지원 (높은 금액, 실패해도 손해 없음)
