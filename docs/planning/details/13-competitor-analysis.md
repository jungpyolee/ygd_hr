## 2. 워키도키 (하이어엑스)

### 2-1. 제품 특징

- **슬로건**: "사장님과 알바님 모두가 편해지는 오프라인 매장관리 필수앱"
- 앱 패키지: `com.clerk.albacheck` / Apple ID: `id1436707534`
- GPS 기반 출퇴근 기록 및 스케줄 관리
- 출퇴근 기반 자동 급여 정산 + 명세서 자동 발송
- **근로계약서 작성** 기능 포함
- 매장 체크리스트 (인증샷 기반 업무 확인 — 원격 관리 가능)
- 교육 매뉴얼 / 공지사항 관리
- **소비기한 관리** (2023.05 정식 출시, 알림 기능 포함)
- **QSC 점검** (품질·서비스·청결) 및 개선 지원
- **매장 점검관리 솔루션** (2024 신규 론칭)
- **무인매장 서비스 "브라우니"** 별도 운영
- 위치 기반 주변 알바 구인구직 (알바생 기능)

### 2-2. 강점

- 단순하고 직관적인 UI — 처음 써도 바로 이해 가능 (App Store 평점 4.3★)
- 사장이 방문 없이 원격으로 알바 업무 현황 파악 가능
- 카카오톡 개인 연락 대신 앱 내 공지로 소통 → 알바생 심리적 부담 감소
- 무인매장 운영에 특히 적합
- 근로계약서 기능으로 법적 요건 대응
- **롯데슈퍼 직영점 200곳** 도입 (대형 고객사 확보)
- **8,000개 이상 매장** 도입 (2022년 기준, 현재는 더 증가 추정)
- **2023년 일본 진출** (민관협력 스타트업 지원사업 선정)
- **2025년 하나시스와 MOU** 체결 → POS 단말 연동 통합 매장관리 솔루션 공동 개발 예정

### 2-3. 약점 (출첵 관점)

- 크레딧/티어 게이미피케이션 **전무**
- 포터블 근태 이력서 개념 **없음**
- 직원 동기부여 요소 **없음** — 관리 도구에 머묾
- 요금제 가격 비공개 (비즈니스 문의 방식)
- 임직원 약 3명 — 개발 속도 느릴 수 있음

### 2-4. 가격

- 구체적 가격표 웹 공개 없음 — 기업 문의 방식 (`support@higherx.co.kr`)
- 과거 정부 비대면 바우처로 70% 할인 이력 있음
- 무료 플랜 여부 불명확

## 3. 샵솔 (위솝)

### 3-1. 제품 특징

- **슬로건**: "소상공인 올인원 HR 솔루션", "직원관리의 핵심"
- GPS 기반 출퇴근 기록 (지각/출근/퇴근 즉시 알림)
- 자동 급여 계산 (4대 보험 공제액 반영)
- **급여명세서 카카오톡 전송** (버튼 한 번)
- 근태 리포트 대시보드 (일별/주별/월별)
- 업무 체크리스트 + 업무일지 작성/공유
- **유통기한 관리** (20만 개 바코드 DB 포함, 조기경보)
- PC 웹 관리자 (급여대장, 근무기록 관리, 엑셀 다운로드)
- 멀티 매장 지원
- **CU편의점 전용 파생앱 "퇴근해CU"**, 코레일유통 전용 "코유매니저" 별도 개발
- 중소규모 프랜차이즈 전용 "세미 커스터마이징 샵솔" (2021.06 서비스 개시)
- **장기 비전**: 비금융 데이터 기반 대안신용평가 시스템 구축 (소상공인 신파일러 문제 해결)

> ⚠️ **기존 가정 수정**: POS 연동, 재고 관리 기능은 샵솔의 핵심 기능이 아닌 것으로 확인. 샵솔은 HR/근태/급여/업무관리 특화 서비스.

### 3-2. 강점

- **Google Play 10만+ 다운로드** — 경쟁사 중 가장 큰 설치 기반
- 기본 기능 무료 제공 (출퇴근, 자동 급여 계산, 급여명세서, 업무 관리)
- 급여명세서 카카오톡 즉시 전송 — 사장님 실무 편의성 높음
- 어르신 사장님도 사용 가능한 단순한 UX
- **BGF리테일(CU), 코레일유통, 이성당** 등 대형 파트너사 확보
- 프랜차이즈 커스터마이징 서비스 — B2B 상위 고객사 대응 가능
- 유통기한 관리 — F&B 업종에 실질적 가치

### 3-3. 약점 (출첵 관점)

- 크레딧/게이미피케이션 **전무**
- 직원 입장에서 "쓰고 싶은 앱"이 아니라 **사장이 쓰라고 해서 쓰는 앱**
- 바이럴 구조 없음 — B2B 영업 의존
- 포터블 근태 이력서 개념 없음
- 크레딧 기반 채용 연계 구상 없음 (단, 비금융 신용평가 비전은 방향이 유사)

### 3-4. 가격

- **기본 기능 무료**: 출퇴근, 자동 급여 계산, 급여명세서 교부, 업무 관리
- **유료 플랜**: PC 웹 관리자 등 프리미엄 기능은 유료 추정 (구체적 금액 공개 없음)
- 프랜차이즈 커스터마이징은 별도 B2B 계약

## 7. 경쟁 전략 제안

### 7-1. 단기 (베타~정식 출시)

```
"워키도키/샵솔이 안 하는 것"에 집중:
  → 크레딧 시스템 = 핵심 차별점으로 마케팅
  → "직원이 좋아하는 출퇴근 앱" 포지셔닝
  → 무료 플랜으로 진입장벽 낮추기
  → 카카오 딥링크 바이럴로 자연 성장
```

### 7-2. 중기 (6~12개월)

```
경쟁사 기능 따라잡기 (필요한 것만):
  → 매장 체크리스트 강화 (워키도키 수준)
  → 전자 근로계약서 (법적 요건)
  → 다점포 관리 강화 (샵솔 대응)
```

### 7-3. 장기 (12개월+)

```
크레딧 기반 독자 생태계:
  → 포터블 크레딧 = 채용 시장 연계
  → "출첵 다이아몬드 티어 → 우대 채용"
  → 샵솔의 "비금융 신용평가" 비전과 유사하지만
    직원 입장에서의 크레딧으로 차별화
```

## 8. 핵심 메시지 (경쟁사 대비)

### vs 워키도키

> "출퇴근 체크하고 공지 올리는 앱? 직원 동기부여까지."
> → 크레딧 시스템으로 직원이 스스로 출근 관리하고 자부심까지

### vs 샵솔

> "유통기한, 업무일지, 체크리스트… 필요한 건 알지만."
> → 직원이 앱을 열 이유가 생기는 건 크레딧과 티어뿐

## 10. 요약

**출첵의 가장 큰 무기는 경쟁사 누구도 갖고 있지 않은 "전역 크레딧 시스템"이다.**

워키도키는 **운영 도구 측면**에서, 샵솔은 **규모와 설치 기반**에서 앞서 있다. 그러나 두 서비스 모두 "직원이 자발적으로 쓰고 싶은 앱"은 아니다. 근태 앱은 사장이 쓰라고 해서 쓰는 구조 — 출첵은 이 공식을 깨는 유일한 서비스를 목표로 한다.

직원이 스스로 쓰고 싶어하는 앱 → 사장님이 도입하기 쉬움 → 크레딧 카드 공유로 자연 바이럴 → 포터블 근태 이력서로 장기 락인.

이 순환 구조는 워키도키/샵솔 어디에도 없다.
