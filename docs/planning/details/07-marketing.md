# 07. 출첵 마케팅 전략

> **작성일**: 2026-03-25
> **버전**: v1.0
> **전제**: 1인 개발자(이정표)가 운영하는 부트스트랩 SaaS. 마케팅 팀 없음. 예산 최소화.
> **핵심 자산**: 연경당 카페에서 2년+ 실운영한 검증된 제품

---

## 1. 타겟 고객 페르소나

### 1-1. 1차 타겟: 소규모 카페 사장님 (직원 5~15명)

| 항목 | 내용 |
|------|------|
| **대표 인물** | 김사장 (34세), 서울 마포구 카페 운영 3년차 |
| **매장 규모** | 카페 1개, 직원 8명 (풀타임 2 + 파트타임 6) |
| **월 매출** | 2,500~4,000만원 |
| **IT 수준** | 카카오톡, 인스타그램은 능숙. 엑셀은 기본 수준 |
| **출퇴근 관리** | 카톡 단톡방에 "출근했어요" 메시지 or 종이 출근부 |
| **급여 정산** | 매달 엑셀로 직접 계산. 2~3시간 소요. 실수 빈번 |
| **스케줄 관리** | 카톡으로 조율. "이번주 누가 언제 나오더라?" 혼란 반복 |

**핵심 Pain Points:**
- 매달 급여일 전 3일이 가장 스트레스. 출근기록 취합 + 엑셀 계산 + 검증
- 알바생이 지각해도 몇 분인지 정확히 모름. 감정적으로 대충 넘김
- 스케줄 변경 요청이 카톡으로 오면 혼선 발생. "내가 그때 바꿔줬는데?"
- 직원 교체 시 이전 매장 근태 확인 불가. 면접에서 "성실해요"라는 말만 믿음

**현재 대안:**
- 카카오톡 단톡방 + 종이 출근부 (무료, 하지만 정리 안 됨)
- 엑셀 급여 템플릿 (인터넷에서 다운. 매달 수작업)
- 알바천국/알바몬 스케줄 기능 (사용법 복잡, 직원들이 잘 안 씀)

**전환 계기:**
- 급여 정산에서 실수로 10만원 더/덜 줌 → "이걸 자동으로 할 수 없을까?"
- 알바생 지각이 반복되는데 객관적 데이터가 없어서 말하기 어려움
- 주변 사장님이 "나 이거 쓰는데 편해" 라고 추천

### 1-2. 2차 타겟: 중형 음식점 사장님 (직원 15~50명)

| 항목 | 내용 |
|------|------|
| **대표 인물** | 박사장 (42세), 강남 한식당 2개 지점 운영 |
| **매장 규모** | 2개 지점, 직원 총 28명 |
| **월 매출** | 8,000만~1.5억원 |
| **IT 수준** | 포스기, 배달앱 사용. 관리 프로그램은 꺼림 |
| **출퇴근 관리** | 지문인식기 있으나 고장 잦음. 결국 수기 보정 |
| **급여 정산** | 세무사에게 위임하지만, 출근 데이터는 본인이 제공해야 함 |

**핵심 Pain Points:**
- 2개 지점 직원 스케줄을 동시에 관리하는 게 지옥
- 지점 간 직원 이동(파견) 시 어디서 몇 시간 일했는지 추적 안 됨
- 지문인식기 유지비가 아까움 (기기 렌탈 + 소프트웨어 월 사용료)
- 직원 30명 급여 정산에 세무사 비용 월 50만원+, 출근 데이터는 여전히 본인이 정리

**현재 대안:**
- 지문인식 출퇴근기 (월 3~5만원 렌탈. 기기 고장, SW 불편)
- 세무사/노무사 위임 (월 30~80만원, 출근 데이터만큼은 직접 제공)
- 식당 전용 POS 연동 근태 (비쌈, 커스터마이징 어려움)

**전환 계기:**
- 지문인식기 계약 만료 시점 → "스마트폰으로 대체할 수 없을까?"
- 2호점 오픈 → 두 매장 직원을 통합 관리할 도구 필요
- 직원에게 급여 명세를 보여달라는 요청 → 수기로 만들기 귀찮음

### 1-3. 3차 타겟: 다점포 체인 (3개+ 매장)

| 항목 | 내용 |
|------|------|
| **대표 인물** | 최대표 (48세), 커피 프랜차이즈 직영 5개점 운영 |
| **매장 규모** | 5개 매장, 직원 총 60명 |
| **IT 수준** | ERP 사용 경험 있음. 효율을 위해 투자 의향 있음 |

**핵심 Pain Points:**
- 매장별 점장(매니저)에게 근태 관리를 위임하지만, 보고 형식이 제각각
- 매장 간 인력 재배치 시 데이터 단절
- 전체 인건비 현황을 한눈에 보고 싶음
- 직원 교체율이 높아 신규 직원 온보딩 비용이 큼

**현재 대안:**
- 시프티, 알밤 등 기존 SaaS (월 직원당 2,000~5,000원. 50명이면 10~25만원)
- 자체 ERP (도입비 500만원+, 유지비 월 50만원+)

**전환 계기:**
- 기존 솔루션 비용 부담 → Free/Starter 플랜으로 시작해보기
- 크레딧 시스템 → 직원 동기부여 + 이직 시 근태 평판 확인 가능
- 카카오 딥링크 초대 → 직원 온보딩이 10초면 끝남

## 2. 포지셔닝 & 메시지

### 2-1. 한 줄 포지셔닝

> **"카페/식당 사장님을 위한 출퇴근 관리 앱"**

부가 설명:

> GPS 출퇴근 + 자동 급여 계산 + 직원 스케줄, 카카오톡으로 5분 만에 시작하세요.

### 2-2. 핵심 가치 3가지

| 순서 | 가치 | 사장님의 말로 |
|------|------|--------------|
| 1 | **시간 절약** | "급여 정산에 3시간 쓰던 게 5분이면 끝나요" |
| 2 | **급여 자동화** | "출근 기록이 자동으로 급여에 반영되니까 실수가 없어요" |
| 3 | **직원 동기부여** | "크레딧 점수 때문에 알바생들이 알아서 안 늦어요" |

### 2-3. 경쟁 대비 차별점

| 비교 항목 | 출첵 | 시프티/알밤 | 카톡 단톡방 | 지문인식기 |
|-----------|------|------------|-------------|-----------|
| **초기 비용** | 무료 (Free 플랜) | 월 직원당 2,000~5,000원 | 무료 | 기기 50~100만원 |
| **직원 온보딩** | 카카오 링크 1번 | 앱 설치 + 가입 + 코드입력 | 이미 사용중 | 지문 등록 방문 |
| **GPS 출퇴근** | O | O | X | X |
| **급여 자동 계산** | O (2대보험/3.3% 자동) | 일부 유료 | X | X |
| **크레딧/근태이력서** | O (업계 유일) | X | X | X |
| **다중 매장** | O (Starter부터) | O (유료) | X | 매장별 기기 필요 |
| **카카오 생태계** | 로그인+초대+공유 통합 | 부분 지원 | 네이티브 | X |
| **F&B 특화** | O (레시피/체크리스트) | 범용 | X | X |

### 2-4. 메시지 프레임워크

**대상별 핵심 메시지:**

| 대상 | 메시지 | 강조점 |
|------|--------|--------|
| 카페 사장님 | "카톡으로 출근부 관리하고 계세요? 출첵이면 5분이면 끝나요" | 전환 용이성, 무료 |
| 음식점 사장님 | "급여일마다 엑셀 붙잡고 계세요? 출퇴근이 자동으로 급여가 돼요" | 급여 자동화, 시간 절약 |
| 알바생 | "내 근태 점수가 스펙이 돼요. 출첵 크레딧으로 증명하세요" | 크레딧, 자기 PR |
| 다점포 대표 | "5개 매장 직원 60명, 한 화면에서 관리하세요" | 통합 대시보드, 비용 절감 |

## 3. 런칭 전략 (베타)

### 3-1. 연경당 성공 사례 활용

연경당은 출첵의 최강 마케팅 자산이다. 2년+ 실운영 데이터가 있다.

**활용 가능한 수치 (예시):**
- "2년간 출퇴근 기록 ○○,○○○건 처리"
- "월 급여 정산 시간 3시간 → 5분으로 단축"
- "직원 지각률 ○○% 감소 (크레딧 도입 후)"
- "직원 ○○명이 매일 사용 중"

**콘텐츠화:**
- 블로그 포스트: "카페 운영 2년, 출퇴근 관리를 이렇게 바꿨습니다"
- 인스타그램 카드뉴스: Before(카톡 출근) vs After(출첵 대시보드) 비교
- 유튜브 숏츠: "우리 카페 직원들이 출근할 때" (실제 GPS 출근 장면)

### 3-2. 초기 100개 매장 확보 전략

#### Phase 1: 시드 유저 (0 → 20개 매장) — 1~2개월

| 전략 | 방법 | 목표 |
|------|------|------|
| **연경당 네트워크** | 연경당 사장님(연경이) 지인 소개, 연경당 거래처 카페/식당 | 5개 |
| **개인 네트워크** | 정표 지인 중 F&B 사장님, 대학 동기, 지역 커뮤니티 | 5개 |
| **소상공인 온라인 커뮤니티** | 네이버 카페(카페창업, 요식업창업), 아프리카/유튜브 사장님 채널 댓글/DM | 5개 |
| **카카오 오픈채팅방** | "카페사장모임", "자영업자모임" 등 기존 방에서 베타 모집 | 5개 |

**시드 유저 혜택:**
- 정식 출시 후 12개월 무료 (Starter 플랜)
- "창립 멤버" 배지 (프로필에 영구 표시)
- 기능 요청 우선 반영 (월 1회 1:1 피드백 미팅)
- 대표 사례 인터뷰 참여 시 추가 3개월 무료

#### Phase 2: 초기 성장 (20 → 50개 매장) — 2~3개월

| 전략 | 방법 |
|------|------|
| **시드 유저 추천** | 사장님 → 사장님 추천. 추천 리워드 제공 |
| **블로그 SEO** | "카페 출퇴근 관리", "알바 급여 계산기", "소상공인 근태관리" 키워드 공략 |
| **인스타그램** | 카페 사장님 인플루언서(팔로워 1만~5만) 협업 (제품 무료 제공) |
| **직원 바이럴** | 크레딧 카드 카카오 공유 → 직원 친구들이 자기 매장 사장님에게 추천 |

#### Phase 3: 확장 (50 → 100개 매장) — 3~4개월

| 전략 | 방법 |
|------|------|
| **콘텐츠 마케팅 본격화** | 주 2회 블로그 + 주 3회 인스타 + 격주 유튜브 |
| **오프라인 참여** | 소상공인 박람회 부스, 상공회의소 세미나 발표 |
| **PR** | 스타트업 미디어(플래텀, 벤처스퀘어) 기고/인터뷰 |
| **제휴** | POS 업체, 배달앱, 세무사 사무소와 상호 추천 |

### 3-3. 베타 테스터 모집 채널

## 4. 채널별 전략

### 4-1. 카카오톡 채널 / 오픈채팅방

**카카오톡 채널 (공식 계정):**
- 채널명: "출첵 - 카페/식당 출퇴근 관리"
- 용도: 공지, 업데이트 알림, 1:1 상담
- 비용: 무료 (발송 건당 과금이지만 초기에는 소량)

**운영 전략:**
- 주 1회 꿀팁 메시지 발송 (급여 계산, 알바 관리 노하우)
- 신규 기능 출시 시 안내
- 장애/점검 공지
- 사장님 Q&A 응대

**카카오 오픈채팅방 활용:**
- 자체 운영: "출첵 사장님 모임" 방 개설 → 사용자끼리 노하우 공유
- 기존 방 참여: "카페사장모임", "소상공인모임" 등에서 자연스럽게 활동
  - 바로 홍보하지 않음. 먼저 질문에 답변하고, 전문성 확보 후 자연스러운 언급
  - "저는 이런 앱 만들어서 쓰고 있는데..." 형태의 소프트 어프로치

## 6. 바이럴/추천 프로그램

### 6-1. 직원 초대 = 자연 바이럴

출첵의 구조적 강점: **사장님이 직원을 초대하는 순간 바이럴이 시작된다.**

```
사장님 가입 → 직원 8명 초대 (카카오 딥링크)
  → 직원 8명의 카카오톡에 "출첵" 노출
  → 직원 중 3명이 다른 매장에서도 알바
  → "우리 매장도 이거 쓰면 좋겠다" → 그 매장 사장님에게 추천
  → 새 매장 사장님 가입 → 직원 10명 초대
  → 반복
```

**이 구조를 강화하는 장치:**
- 직원 앱 하단에 "우리 매장도 출첵 쓰기" 배너 (사장님 가입 유도)
- 크레딧 카드를 카카오로 공유할 때 출첵 브랜딩 자연 노출
- 직원이 다른 매장에 합류할 때 → 기존 크레딧 유지 → "이 앱 뭐예요?" 대화 유발

### 6-2. 사장님 → 사장님 추천 프로그램

**"추천하면 둘 다 좋은" 프로그램:**

| 항목 | 추천인 (기존 사장님) | 피추천인 (새 사장님) |
|------|---------------------|---------------------|
| 보상 | Starter 1개월 무료 연장 | Starter 1개월 무료 체험 |
| 조건 | 피추천인이 14일 이상 활성 사용 | 가입 후 직원 3명 이상 초대 |
| 제한 | 월 최대 3건 (악용 방지) | - |

**추천 방법:**
- 관리자 페이지 → "사장님 친구에게 추천하기" 버튼
- 고유 추천 코드 생성 (예: REF-YEONGYEONG-001)
- 카카오톡 공유 → "저도 쓰고 있는 출퇴근 앱이에요" 메시지 + 추천 링크

### 6-3. 가맹점주 네트워크 효과

프랜차이즈 가맹점주 모임은 강력한 바이럴 채널이다.

**시나리오:**
- 커피 프랜차이즈 A의 가맹점주 1명이 출첵 도입
- 가맹점주 월례회의에서 "나 이거 쓰는데 편해" 언급
- 다른 가맹점주 3~5명이 관심 → 시도
- 본사가 공식 도입 검토

**대응 전략:**
- Pro 플랜에 "본사 대시보드" 기능 (모든 가맹점 통합 조회)
- 프랜차이즈 전용 Enterprise 플랜 (향후 확장)
- 가맹점주 모임 발표/시연 제안

### 6-4. 추천 리워드 누적 체계

```
1명 추천: Starter 1개월 무료
3명 추천: Starter 2개월 무료 + "추천왕" 배지
5명 추천: Pro 1개월 무료
10명 추천: Starter 6개월 무료 + 1:1 컨설팅
```

## 7. 크레딧 시스템 마케팅

크레딧/티어 시스템은 출첵만의 고유 무기이다. 경쟁사에 없는 기능이다.

### 7-1. "근태 이력서" 개념 홍보

**사장님 대상:**
- "면접 볼 때 크레딧 점수 보여달라고 하세요. 성실한 직원인지 바로 알 수 있어요."
- "크레딧 다이아 직원이 지원하면 채용 확률 높이세요."
- "직원이 알아서 안 늦는 마법: 크레딧이 쌓이니까요."

**알바생 대상:**
- "출첵 크레딧 = 근태 스펙. 다음 알바 면접에서 꺼내 보세요."
- "지각 0, 출근 100일 연속 → 다이아몬드 티어. 이력서보다 강력해요."
- "크레딧 카드 캡처해서 면접관한테 보여주세요."

### 7-2. 알바 커뮤니티 연계 가능성

| 플랫폼 | 연계 방식 | 단계 |
|--------|----------|------|
| **알바몬** | 이력서에 "출첵 크레딧 점수" 연동 필드 제안 | 중장기 (MAU 1만+ 이후) |
| **알바천국** | 구직 프로필에 크레딧 티어 표시 제안 | 중장기 |
| **에브리타임** | 대학생 알바 커뮤니티에서 크레딧 자랑 → 자연 홍보 | 단기 (커뮤니티 활동) |
| **블라인드** | "직장인 근태 점수" 밈 활용 → 인지도 확대 | 단기 (밈/콘텐츠) |

**단기 실행 가능한 것:**
- 에브리타임/대학 커뮤니티에 "알바 근태 점수 인증" 글 유도
- "크레딧 다이아 달성 인증" 이벤트 → SNS 공유 시 치킨 기프티콘 추첨

### 7-3. 크레딧 카드 공유 = 자연스러운 앱 노출

크레딧 카드를 카카오톡으로 공유하면 다음 정보가 포함된다:

```
┌──────────────────────────────┐
│  출첵 근태 프로필            │
│  김민수 | 다이아몬드 티어     │
│  크레딧 점수: 920            │
│  정시 출근율: 96.8%          │
│  총 근무일수: 342일          │
│  ────────────────────────── │
│  출첵 | chulchek.app         │
└──────────────────────────────┘
```

**이 카드를 공유할 때마다:**
- 받는 사람이 "출첵이 뭐야?" → 앱 인지도 상승
- 사장님이 받으면 "우리 매장도 도입해볼까?" → 직접 전환
- 다른 알바생이 받으면 "나도 크레딧 쌓고 싶다" → 직원이 사장님에게 추천

**공유 유도 장치:**
- 티어 승급 시 축하 메시지 + "공유하기" 버튼 강조
- 월간 리포트 → "이번 달 크레딧 +45점! 친구에게 자랑하기"
- 크레딧 카드 디자인을 계절/이벤트별로 변경 → 수집/공유 욕구 자극

## 9. KPI & 마일스톤

### 9-1. 핵심 KPI

| KPI | 정의 | 측정 방법 |
|-----|------|-----------|
| **MAO** (Monthly Active Organizations) | 월 1회 이상 출퇴근 기록이 있는 조직 수 | DB 쿼리 |
| **MAU** (Monthly Active Users) | 월 1회 이상 로그인한 사용자 수 | Supabase Auth 로그 |
| **매장 등록 수** | 누적 조직 생성 수 | organizations 테이블 |
| **직원 등록 수** | 누적 멤버십 수 | organization_memberships |
| **DAU/MAU 비율** | 일간 활성/월간 활성 비율 (stickiness) | 40% 이상 목표 |
| **NPS** | 순추천점수 | 분기별 설문 |
| **전환율** | Free → Starter 전환 | 결제 데이터 |
| **Churn Rate** | 월간 이탈률 | 30일 미접속 조직 |
| **CAC** | 고객 획득 비용 | 마케팅 비용 / 신규 매장 수 |
| **ARPU** | 사용자당 평균 매출 | 월 매출 / 유료 조직 수 |
