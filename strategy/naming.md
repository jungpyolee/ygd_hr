## 후보 목록

### 브랜드감
| 이름 | 느낌 |
|------|------|
| Tempo | 시간의 리듬, 깔끔 |
| Ponto | 위치/점, 글로벌 |
| Attendo | Attend+o, 출퇴근 |

### 심플 영어
| 이름 | 느낌 |
|------|------|
| 체크온 (CheckOn) | 체크+온, 직관적 |
| 탭인 (TapIn) | 출퇴근 탭, 중립적 |

### 한국어 친근 스타일 (유력)
| 이름 | 느낌 |
|------|------|
| **출근이요** | 출근=시작, 앞으로 나아가는 느낌 |
| **퇴근이요** | 재밌고 기억에 남음, 역설적으로 눈에 띔 |
| **근태이요** | 출퇴근 전체 포괄, 중립 |
| 출근요 | 직관적, 깔끔 |
| 찍어요 | 찍는 행동, 재밌음 |
| 근태요 | 친근하지만 "근태" 단어가 살짝 딱딱 |
