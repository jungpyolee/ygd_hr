## 1. 가격 전략

### 월 2만원 단일 요금제는 현실적으로 힘든 이유
- 카톡/밴드/수기 등 무료 대안이 이미 있음
- 소상공인은 직접적인 매출 연결이 안 보이면 지갑 안 열음
- "굳이?" 반응이 기본값

### 현실적인 가격 구조

**방법 1 — 프리미엄 (추천)**
```
무료: 직원 3명 이하
유료: 4명 이상 → 월 9,900원
```
"커피 두 잔" 프레이밍 가능

**방법 2 — 인당 과금**
```
1인당 월 1,500원
5명 → 7,500원 / 10명 → 15,000원
```
규모 클수록 자연스럽게 올라감

**방법 3 — 연간 결제 할인**
```
월 9,900원 or 연 79,000원 (2달 무료)
```
이탈률 낮추는 효과

## 2. ROI — 실제로 새는 돈

### 항목별 손실

| 항목 | 내용 | 월 손실 |
|------|------|---------|
| 출퇴근 조작 | 10분 일찍 퇴근 기록 × 알바 3명 | 99,000원 |
| 주휴수당 과지급 | 14시간 → 15시간으로 기록, 주휴수당 발생 | 50,000원 |
| 초과근무 허위 | "오늘 30분 더 있었어요" 확인 불가 | 30,000원 |
| **합계** | | **월 179,000원** |
| **연간** | | **2,148,000원** |

**알바 1명 한 달치 월급이 그냥 사라지는 중.**

### 추가 절약 항목

- **주휴수당 오계산 방지**: 미지급 시 노동부 신고 → 체불임금 + 과태료 수백만원
- **노무 분쟁 시 GPS 기록이 법적 증거**: 노무사 선임 비용 30~100만원 방어
- **스케줄 최적화**: 한산한 시간대 인력 낭비 월 10~20시간 감축 → 월 100,000~200,000원 절약

### 손익 비교

|------|----------------|---------|

## 3. 핵심 메시지 & 카피

### 포지셔닝
**"근태 관리 앱"이 아니라 "인건비 새는 거 막는 앱"**

### 카피 후보

```
"알바 3명이면 매달 10만원씩 새고 있을 수 있어요.
 월 9,900원으로 막으세요."

"사장님, 알바가 월급 외에 매달 17만원을 더 가져가고 있어요."

"CCTV는 달았는데 출퇴근은 알바 말만 믿고 계세요?"

"알바는 1분도 안 빠뜨리는데
 사장님은 매달 몇 시간씩 빠지고 있어요."
```

### 감정 코드 (구매 결정의 핵심)
```
억울함 → "내가 더 열심히 일하는데"
배신감 → "믿었던 알바가"
절박함 → "마진이 얼마나 남는다고"
```

기능 설명보다 감정이 먼저 움직여야 지갑이 열린다.
